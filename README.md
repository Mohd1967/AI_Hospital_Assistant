# AI Hospital Assistant

مشروع مساعد طبي ذكي يعتمد على:
- بوت تيليجرام
- أتمتة Medical Automation
- ChatGPT / AI
- نظام استشارات طبية عن بعد
- ربط مع دستور الأدوية (الأمريكي – الأوروبي – العربي – العالمي)
- موقع وتطبيق خاص بالمستشفى
- محاضرات تعليمية لطلاب الطب، الأسنان، الصيدلة، التمريض

## أهداف المشروع
- تسهيل الوصول للمعلومات الطبية الصحيحة.
- دعم الأطباء والطلاب بقاعدة معرفية قوية.
- تقديم استشارات طبية أولية بطريقة آمنة وفعّالة.
- دعم مشروع **مدينة أمي الطبية**.
